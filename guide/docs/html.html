HTML 규칙
들여쓰기
  • 1탭 들여쓰기 하며, 1탭의 크기는 공백 4칸으로 설정한다.
빈줄
  • 가독성을 고려하여 빈 줄을 추가한다. 단, 1줄을 초과하지 않는다.
주석
  • 기본형식 <!-- mod-date:0201: 주석내용 --><!-- //주석내용 -->
특수기호 Entity name
  • 특수기호(<, >, &)는 Entity name을 사용하여 entity 코드로 변환한다.

노치디자인(아이폰X Safe Area) 대응
  • Viewport 설정
    <meta name="viewport" content="width=device-width, initial-scale=1.0, viewport-fit=cover">
  • 인디케이터 영역 설정
    iOS 11 constant() / iOS 11.2 이상 env() 사용
    // iOS 11.0 버전 constant(safe-area-inset-top) constant(safe-area-inset-right) constant(safe-area-inset-bottom) constant(safe-area-inset-left) // iOS 11.2 이상 env(safe-area-inset-top) env(safe-area-inset-right) env(safe-area-inset-bottom) env(safe-area-inset-left)
  • iPhoneX(v12.3.1) 세로모드
    .btn { padding:15px 0 calc(constant(safe-area-inset-bottom) + 15px); padding:15px 0 env(constant(safe-area-inset-bottom) + 15px); }
  • iPhoneX(v12.3.1) 가로모드
    .wrap { padding-right:constant(safe-area-inset-right) + 15px); padding-left:constant(safe-area-inset-left) + 15px); padding-right:env(safe-area-inset-right) + 15px); padding-left:env(safe-area-inset-left) + 15px); } .btn { padding:15px 0 calc(constant(safe-area-inset-bottom) + 15px); padding:15px 0 env(constant(safe-area-inset-bottom) + 15px); }
