Selection Control

사용자가 선택 옵션들을 제어할 수 있도록 돕는 컴포넌트
Type
CheckBox
옵션명은 2줄로 노출하지 않는다
옵션명이 영역을 초과할 경우 말줄임표(...)를 노출할 수 있으나 지양한다
large(24x24) / medium(20x20)
Radio Button
두 개 이상의 항목에서 하나만 선택할 수 있는 경우에 사용
선택되지 않은 Radio Button을 선택하는 순간, 이전에 선택한 항목은 자동으로 선택이 해제된다.
large(24x24) / medium(20x20)
Switch
각 선택 항목의 상태를 즉시 제어할 때 사용
large(24x24) / medium(20x20)
