Swiper

1. 기본사용법

  • 기본 swiper 마크업에 클래스추가 또는 data 타입 선언으로 swiper option 적용

2. swiper update

  • UICommon.uiSwiper.update('셀렉터')

3. swiper destroy

  • UICommon.uiSwiper.destroy('셀렉터')

4. swiper create

  • UICommon.uiSwiper.init('셀렉터'), 셀렉터 없을 경우 전체 create
